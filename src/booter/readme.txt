The game is Space Invaders. This is how to play:

Keys for interactivity:
    Left: move player left
    Right: move player right
    Spacebar: player fires missile/bullet
    'r': reset game manually
    'p': pause/un-pause game
    
Progress-bar color significance:
    Blue: game is in-play
    Gray: game is paused
    White: game is being reset manually ('R' key)
    Red: player lost game
    All green: player won game
